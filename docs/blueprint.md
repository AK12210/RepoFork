# **App Name**: RepoFork

## Core Features:

- Repository Cloning: Clone a GitHub repository by providing its URL.
- File Browsing: Browse the contents of the cloned repository, including files and directories.
- README/Contributing Display: Automatically detect and display the README.md or CONTRIBUTING.md file (if present) in a formatted view.
- Summary Generation: Generate a summary of the repo's contents, highlighting key files, technologies used, and contribution guidelines using a generative AI tool.
- Tech Stack Identification: Automatically detect the primary programming languages, frameworks, and libraries used in the repository.
- Contribution Guideline Extraction: Attempt to extract the main guidelines from a CONTRIBUTING.md or similar file, and present them in an easy-to-digest format.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to evoke trust and stability.
- Background color: Light gray (#F0F4F8), near-white, for a clean and readable layout.
- Accent color: Teal (#26A69A) for interactive elements and highlights to contrast the primary color.
- Font pairing: 'Inter' (sans-serif) for body text to ensure readability and 'Space Grotesk' (sans-serif) for headlines for a techy feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple, outlined icons for file types and actions. Consider icons that match github's icon style.
- Clean and structured layout with clear visual hierarchy. Prioritize content discoverability.
- Subtle animations for loading and transitions to improve the user experience.
